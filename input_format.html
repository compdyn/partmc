Spec File Format

When running PartMC with the command partmc input.spec the first line of the input.spec file must define the run_type with: 

 run_type <type>

where <type> is one of particle, exact, or sectional. This determines the type of run as well as the format of the remainder of the spec file. The rest of the spec file is described by:
