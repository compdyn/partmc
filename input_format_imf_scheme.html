Input File Format: Immersion Freezing Scheme

Read the specification for immersion freezing scheme.

Parameters
[in,out]fileSpec file.
[out]immersion_freezing_scheme_typeImmersion freezing scheme type.

The immersion freezing scheme is specified by the parameter:

  • immersion_freezing_scheme (string): the type of immersion freezing scheme must be one of: const for freezing with a constant freezing rate; singular for the INAS scheme based on singular perspective; ABIFM for the ABIFM scheme based on CNT perspective.

If immersion_freezing_scheme is singular, the parameters INAS_a and INAS_b need to be provided. If immersion_freezing_scheme is const, the parameter freezing_rate needs to be provided. If immersion_freezing_scheme is ABIFM or const, a logical variable do_freezing_naive needs to be provided.
