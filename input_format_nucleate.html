Input File Format: Nucleation Parameterization
Parameters
[in,out]fileSpec file.
[in,out]aero_dataAerosol data.
[out]nucleate_typeNucleate type.
[out]nucleate_sourceNucleate source number.
[out]nucleate_weight_classNucleate weight class.

The nucleation parameterization is specified by the parameter:

  • nucleate (string): the type of nucleation parameterization — must be one of: "none" for no nucleation; or "sulf_acid" for the nucleate_sulf_acid() parameterization
