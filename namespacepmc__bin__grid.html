logical function pmc_bin_grid::bin_grid_contains ( type(bin_grid_t), intent(in)  bin_grid,
integer, intent(in)  i_bin,
real(kind=dp), intent(in)  val 
)

Determines if a value is in a given bin.

The value is contained in bin i if edge(i) <= val < edge(i+1). If bin_i = 0, then we return true if val < edge(1). If bin_i = bin_grid_size + 1, then we return true if val > edge(bin_grid_size+1). If val = edge(bin_grid_size+1), then return true if i_bin = bin_grid_size.

Parameters
[in]bin_gridBin grid.
[in]i_binBin to see if it contains the value.
[in]valData value.

real(kind=dp), intent(in)  val 
)

Find the bin number that contains a given value.

If a particle is below the smallest bin then its bin number is 0. If a particle is above the largest bin then its bin number is bin_grid_size(bin_grid) + 1.

Parameters
[in]bin_gridBin_grid.
[in]valValue to locate bin for.
